## 2026-04-05 - Keyboard Focus Outlines on Custom Block Elements
**Learning:** Default browser focus rings are often clipped, obscured, or visually confusing when applied to custom block-level interactive components (like cards or rows used as buttons).
**Action:** Explicitly define `:focus-visible` styles with a clear outline and `outline-offset` using the theme's accent color for all interactive elements to ensure reliable keyboard navigation visibility.
