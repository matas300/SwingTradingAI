## 2026-04-14 - Missing keyboard focus states\n**Learning:** In a vanilla CSS project without a reset library, it's very easy to fail to provide a visible focus state for interactive elements (like buttons, links, inputs). Keyboard users require a strong visual focus indicator to navigate.\n**Action:** Add explicit `:focus-visible` styles using the existing `--accent` design token to all interactive elements (`a`, `button`, `input`, `select`, `textarea`) to guarantee accessible keyboard navigation.
